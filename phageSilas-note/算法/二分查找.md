# 基础
``` java
public static int binarySearchBasic(int[] arr, int target) {  
    int i=0, j=arr.length-1;  
    while(i<=j){  
       // int m=(i+j)/2;//会自动向下取整,索引号  
        int m=(i+j) >>> 1; // 位运算，等同于 (i+j)/2,防止溢出int  
  
        if(target < arr[m]){  
            j=m-1;//目标在左边,将有边界向左移  
        }else if(arr[m] < target ){//都写成小于方便理解  
            i=m+1;//目标在有边,将左边界向右移  
  
        }else{  
            return m;  
        }  
  
    }  
    return -1;  
}
```
# i>>>1
位运算,二进制向右移一位,等效于除以二,且和普通运算一下,**会自动向下取整**
可防止int类型溢出
