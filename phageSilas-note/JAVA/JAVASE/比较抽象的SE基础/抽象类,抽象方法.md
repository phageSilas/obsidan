# 1.抽象类和接口的区别

| 特性       | 普通类      | 抽象类            | 接口                                          |
| -------- | -------- | -------------- | ------------------------------------------- |
| **实例化**  | 可以直接实例化  | ❌ 不能实例化        | ❌ 不能实例化                                     |
| **方法实现** | 所有方法都有实现 | **可以有抽象和具体方法** | Java 8前：全部**抽象**  <br>Java 8+：可有**默认/静态方法** |
| **继承**   | 单继承      | 单继承            | 多继承（一个类可实现多个接口）                             |
| **构造方法** | 有构造方法    | **有构造方法**      | 没有构造方法                                      |
| **成员变量** | 任意类型     | 任意类型           | 只能是常量（public static final）                  |
| **设计目的** | 具体实现     | 部分实现 + 部分规范    | 完全规范                                        |
| **何时使用** | 完整的具体类   | 有共享代码的相关类      | 定义跨继承树的行为                                   |
