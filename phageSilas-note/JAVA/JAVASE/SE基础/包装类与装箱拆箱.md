## 自动拆箱和装箱

### 概念解释

**自动装箱（Autoboxing）**：将基本类型自动转换为对应的包装类
```java
int a = 10;
Integer b = a;  // 自动装箱：int → Integer
```


**自动拆箱（Unboxing）**：将包装类自动转换为对应的基本类型
```java
Integer b = 10;
int a = b;  // 自动拆箱：Integer → int
```


### Java 的 8 种基本类型和包装类

| 基本类型 | 包装类 |
|---------|--------|
| `int` | `Integer` |
| `long` | `Long` |
| `double` | `Double` |
| `float` | `Float` |
| `boolean` | `Boolean` |
| `byte` | `Byte` |
| `short` | `Short` |
| `char` | `Character` |

## 什么场景会触发自动拆箱？

### 1️⃣ **赋值给基本类型变量**
```java
Integer num = 100;
int primitive = num;  // 自动拆箱：Integer → int
```


### 2️⃣ **方法返回基本类型**
```java
public boolean test() {
    Boolean flag = null;
    return flag;  // ❌ 自动拆箱时，null → boolean 会抛 NullPointerException
}
```

**这就是你刚才代码中的场景！**

### 3️⃣ **算术运算**
```java
Integer a = 10;
Integer b = 20;
int sum = a + b;  // 自动拆箱后进行加法运算
```


### 4️⃣ **比较运算**
```java
Integer x = 10;
Integer y = 20;
if (x < y) {  // 自动拆箱后比较
    System.out.println("x < y");
}
```


### 5️⃣ **三元运算符**
```java
Integer num = null;
int result = (num != null) ? num : 0;  // num 会被自动拆箱
```


### 6️⃣ **集合操作时的计算**
```java
List<Integer> list = new ArrayList<>();
list.add(10);
list.add(20);
int sum = list.get(0) + list.get(1);  // 自动拆箱
```


### 7️⃣ **switch 语句**
```java
Integer num = 5;
switch (num) {  // 自动拆箱为 int
    case 5:
        System.out.println("five");
        break;
}
```

## ⚠️ 空指针陷阱

**当包装类为 `null` 时，自动拆箱会抛出 `NullPointerException`：**

```java
// ❌ 危险示例
Boolean flag = null;
if (flag) {  // NullPointerException！
    System.out.println("true");
}

// ✅ 安全写法
if (BooleanUtil.isTrue(flag)) {  // null 会被当作 false 处理
    System.out.println("true");
}

// ✅ 或者手动判空
if (flag != null && flag) {
    System.out.println("true");
}
```
