## 1.简要
同一个行为具有多种表现形式的能力
## 2.基本定义
多态（Polymorphism）是面向对象编程的三大特性之一（另外两个是封装和继承）。多态的字面意思是“多种形态”，在Java中，多态指的是同一个行为具有多个不同表现形式或形态的能力。

## 3.必要条件
1. **继承**（或实现接口）
2. **重写**（子类重写父类的方法）
3. **父类引用指向子类对象**（向上转型）

## 4.多态的好处
1. **可替换性**：多态对已存在的代码具有可替换性。  
2. **可扩充性**：增加新的子类不影响已有类的多态性、继承性，以及其他特性的运行和操作。
3. **接口性**：多态是超类通过方法签名，向子类提供了一个共同接口，由子类来完善或者覆盖它而实现的。
4. **灵活性**：在应用中体现了灵活多样的操作，提高了使用效率。
5. **简化性**：多态简化了对应用软件的代码编写和修改过程，尤其在处理大量对象的运算和操作时。

## 5.多态的实现方式：
1. **重写**（Override）：子类对父类的方法进行重新定义，在调用时根据对象的实际类型来执行对应的方法。
2. **重载**（Overload）：在同一类中，方法名相同，参数列表不同，与多态无关（编译时多态，而重写是运行时多态）。

```java
// 1. 继承关系
class Animal {}           // 父类
class Dog extends Animal {} // 子类
class Cat extends Animal {} // 子类

// 2. 方法重写
class Animal {
    public void makeSound() {}  // 父类方法
}

class Dog extends Animal {
    @Override
    public void makeSound() {   // 子类重写
        System.out.println("汪汪汪");
    }
}

// 3. 向上转型
Animal animal = new Dog();  // 父类引用指向子类对象
```
## 6.**多态的核心：向上转型和动态绑定**

### 向上转型

```java

Animal animal = new Dog();  // 向上转型
// 将Dog对象当作Animal类型使用
// 编译时类型：Animal
// 运行时类型：Dog

```
### 动态绑定

```java
public class DynamicBindingDemo {
    public static void main(String[] args) {
        Animal animal = new Dog();  // 向上转型
        
        // 动态绑定：运行时根据实际对象类型调用方法
        animal.makeSound();  // 输出：汪汪汪
        
        // 即使父类引用，实际调用的是Dog的方法
        // 因为JVM在运行时检查对象的实际类型
    }
}
```

**动态绑定过程：**

1. 编译器检查`Animal`类是否有`makeSound()`方法
2. 如果有，编译通过
3. 运行时，JVM查找对象的实际类型（Dog）
4. 调用Dog类的`makeSound()`方法

## **7.多态的四种表现形式**

### 形式1：父类引用指向子类对象

```java
class Shape {
    void draw() { System.out.println("绘制形状"); }
}

class Circle extends Shape {
    @Override
    void draw() { System.out.println("绘制圆形"); }
}

class Square extends Shape {
    @Override
    void draw() { System.out.println("绘制方形"); }
}

public class Test {
    public static void main(String[] args) {
        Shape shape1 = new Circle();  // 父类引用指向Circle
        Shape shape2 = new Square();  // 父类引用指向Square
        
        shape1.draw();  // 输出：绘制圆形
        shape2.draw();  // 输出：绘制方形
    }
}
```

### 形式2：接口引用指向实现类对象

```java
interface Payment {
    void pay(double amount);
}

class Alipay implements Payment {
    @Override
    public void pay(double amount) {
        System.out.println("支付宝支付：" + amount + "元");
    }
}

class WechatPay implements Payment {
    @Override
    public void pay(double amount) {
        System.out.println("微信支付：" + amount + "元");
    }
}

public class Test {
    public static void main(String[] args) {
        Payment payment = new Alipay();  // 接口引用指向实现类
        payment.pay(100.0);  // 输出：支付宝支付：100.0元
        
        payment = new WechatPay();       // 同样的引用指向不同的实现类
        payment.pay(200.0);  // 输出：微信支付：200.0元
    }
}

```
### 形式3：方法参数多态

```java
class Animal {
    void eat() { System.out.println("动物吃东西"); }
}

class Dog extends Animal {
    @Override
    void eat() { System.out.println("狗吃骨头"); }
}

class Cat extends Animal {
    @Override
    void eat() { System.out.println("猫吃鱼"); }
}

class AnimalFeeder {
    // 方法参数是Animal类型，可以接受任何Animal子类
    public void feed(Animal animal) {
        animal.eat();  // 根据传入的实际对象调用不同的eat方法
    }
}

public class Test {
    public static void main(String[] args) {
        AnimalFeeder feeder = new AnimalFeeder();
        
        feeder.feed(new Dog());  // 输出：狗吃骨头
        feeder.feed(new Cat());  // 输出：猫吃鱼
        feeder.feed(new Animal());  // 输出：动物吃东西
        
        // 一个方法，处理多种类型
    }
}

```
### 形式4：方法返回类型多态

```java
abstract class Vehicle {
    abstract void drive();
}

class Car extends Vehicle {
    @Override
    void drive() { System.out.println("驾驶汽车"); }
}

class Bike extends Vehicle {
    @Override
    void drive() { System.out.println("骑自行车"); }
}

class VehicleFactory {
    // 根据条件返回不同类型的Vehicle
    public Vehicle createVehicle(String type) {
        if ("car".equals(type)) {
            return new Car();    // 返回Car，但类型是Vehicle
        } else if ("bike".equals(type)) {
            return new Bike();   // 返回Bike，但类型是Vehicle
        }
        return null;
    }
}

public class Test {
    public static void main(String[] args) {
        VehicleFactory factory = new VehicleFactory();
        
        Vehicle vehicle1 = factory.createVehicle("car");
        Vehicle vehicle2 = factory.createVehicle("bike");
        
        vehicle1.drive();  // 输出：驾驶汽车
        vehicle2.drive();  // 输出：骑自行车
    }
}
```
## 8. **多态的优点**

### 优点1：提高代码扩展性

```java
// 没有多态：每增加一种动物，就要修改代码
public class ZooWithoutPolymorphism {
    public void makeSound(Dog dog) {
        dog.bark();
    }
    
    public void makeSound(Cat cat) {
        cat.meow();
    }
    
    public void makeSound(Bird bird) {
        bird.chirp();
    }
    // 每增加一种动物，就要增加一个方法
}

// 使用多态：增加新动物，不需要修改现有代码
public class ZooWithPolymorphism {
    public void makeSound(Animal animal) {
        animal.makeSound();  // 所有动物都继承Animal
    }
    // 增加新动物：只需要创建新类继承Animal，不需要修改这个方法
}
```

### 优点2：提高代码复用性

```java
// 通用的排序算法
public class Sorter {
    // 可以排序任何实现Comparable接口的对象
    public void sort(Comparable[] array) {
        // 排序逻辑...
        // 比较时使用：array[i].compareTo(array[j])
        // 不需要知道具体是什么对象
    }
}

// 各种对象实现Comparable接口
class Student implements Comparable<Student> {
    private String name;
    private int score;
    
    @Override
    public int compareTo(Student other) {
        return Integer.compare(this.score, other.score);
    }
}

class Product implements Comparable<Product> {
    private String name;
    private double price;
    
    @Override
    public int compareTo(Product other) {
        return Double.compare(this.price, other.price);
    }
}

// 使用同一个排序方法
public class Test {
    public static void main(String[] args) {
        Sorter sorter = new Sorter();
        
        Student[] students = new Student[10];
        Product[] products = new Product[10];
        
        sorter.sort(students);  // 按分数排序
        sorter.sort(products);  // 按价格排序
        // 同样的sort方法，处理不同的对象
    }
}

```
### 优点3：降低耦合度

```java
// 高层模块不依赖具体实现，只依赖抽象
interface Database {
    void connect();
    void query(String sql);
}

class MySQLDatabase implements Database {
    @Override
    public void connect() {
        System.out.println("连接MySQL数据库");
    }
    
    @Override
    public void query(String sql) {
        System.out.println("执行MySQL查询：" + sql);
    }
}

class OracleDatabase implements Database {
    @Override
    public void connect() {
        System.out.println("连接Oracle数据库");
    }
    
    @Override
    public void query(String sql) {
        System.out.println("执行Oracle查询：" + sql);
    }
}

public class Application {
    private Database database;
    
    // 构造函数注入，不依赖具体实现
    public Application(Database database) {
        this.database = database;
    }
    
    public void run() {
        database.connect();
        database.query("SELECT * FROM users");
    }
    
    public static void main(String[] args) {
        // 可以轻松切换数据库
        Application app1 = new Application(new MySQLDatabase());
        Application app2 = new Application(new OracleDatabase());
        
        app1.run();  // 使用MySQL
        app2.run();  // 使用Oracle
    }
}
```

## 9. **多态的限制与注意事项**

### 限制1：不能访问子类特有方法

```java
class Animal {
    public void eat() { System.out.println("动物吃东西"); }
}

class Dog extends Animal {
    @Override
    public void eat() { System.out.println("狗吃骨头"); }
    
    public void bark() { System.out.println("汪汪汪"); }
}

public class Test {
    public static void main(String[] args) {
        Animal animal = new Dog();
        
        animal.eat();     // ✅ 可以调用
        // animal.bark();  // ❌ 不能调用Dog特有方法
        
        // 需要向下转型
        if (animal instanceof Dog) {
            Dog dog = (Dog) animal;  // 向下转型
            dog.bark();  // ✅ 现在可以了
        }
    }
}
```

### 限制2：静态方法没有多态

```java
class Parent {
    public static void staticMethod() {
        System.out.println("Parent.staticMethod");
    }
    
    public void instanceMethod() {
        System.out.println("Parent.instanceMethod");
    }
}

class Child extends Parent {
    public static void staticMethod() {
        System.out.println("Child.staticMethod");  // 不是重写，是隐藏
    }
    
    @Override
    public void instanceMethod() {
        System.out.println("Child.instanceMethod");  // 重写
    }
}

public class Test {
    public static void main(String[] args) {
        Parent p = new Child();
        
        p.staticMethod();    // 输出：Parent.staticMethod（编译时确定）
        p.instanceMethod();  // 输出：Child.instanceMethod（运行时确定）
    }
}
```

### 限制3：属性没有多态

```java
class Parent {
    int value = 10;
}

class Child extends Parent {
    int value = 20;  // 不是重写，是隐藏
}

public class Test {
    public static void main(String[] args) {
        Parent p = new Child();
        System.out.println(p.value);  // 输出：10（看引用类型）
        
        Child c = new Child();
        System.out.println(c.value);  // 输出：20
    }
}
```
